This style guide sets guidelines for writing materials for W3C Accessibility Guidelines (WCAG 3) and other W3C documents. The purpose of the guide is to support:

* readability,  
* consistency, and  
* better, easier translations.

This is a living document. It aims to follow the recommendations in this style guide.

> [!IMPORTANT] 
> These guidelines are written for the **English language**. Translators may apply regional conventions and or orthotypography.

## Feedback
