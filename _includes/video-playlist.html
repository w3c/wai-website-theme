{% comment %} To use this component you need to include your playlist in the page front matter. If there is transcript information then include that when including the component using the `transcripts` parameter. Structure for the playlist: playlist: - title: youtube-id: <the YouTube id for the video> default-lang: <language code for the default language of the video> captions: <lang code>: <caption file location> . . . subtitles: <lang code>: <subtitles file location> . . . descriptions: <lang code>: <description file location> . . . Transcripts should be added before the include command. They should be of the form: <div id="<youtube id-transcript" class="transcript" style="display: block|none"> <Transcript text> </div> . . . The first transcript should be `display: block`.
